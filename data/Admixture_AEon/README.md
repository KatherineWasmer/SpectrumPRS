### Admixture for LaSalle Dataset (n = 55) 

These are the admixture proportions estimated with [AEon](https://www.biorxiv.org/content/10.1101/2024.06.18.599246v1.full), an open-source tool available for estimating ancestry directly from VCF files. These are simply estimates based on the given HGDP reference samples and should be taken with a grain of salt.
